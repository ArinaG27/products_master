1. В папке requests создать js-файл с запросами для users.

2. Написать fetch-запрос по ссылке https://dummyjson.com/users, выводящий массив с объектами в консоль.

3. Вызвать функцию с fetch-запросом в компоненте App.

4. Создать компоненты Users и UsersContainer.

5. Из объектов из состояния users сформировать карточки внутри UsersContainer. Использовать useContext

6. Сделать общее навигационное меню их двух ссылок - Products и Users

7. Разбить App на две страницы - ProductsPage и UsersPage

8. С помощью библиотеки react-router-dom настроить переключение между страницами через навигационное меню

9. Создать компонент AddUserForm, импортировать его на страницу с юзерами. Форма имеет 3 инпута - firstname, lastname, age

10. Установить в проект библиотку react-hook-form

11. С помощью useForm научить форму собирать данные и выводить их в консоль

12. Сделать все поля формы обязательными для заполнения

13. Настроить вывод сообщения об ошибке в случае, если обязательное поле не заполнено

14. Отправить fetch POST запрос при отправке формы AddUserForm

15. Создать AddProductForm (с использованием useForm)  и отправить fetch POST запрос при отправке формы AddProductForm

16. Создать страницу PostsPage. Вывести посты (GET fetch-запрос), доступные по ссылке https://dummyjson.com/posts - вывести title и body.

17. Создать страницу ToDosPage. вывести задачи (GET fetch-запрос), доступные по ссылке https://dummyjson.com/todos - вывести todo. Если completed true вывести параграф 'task done', если false - 'task not done'
